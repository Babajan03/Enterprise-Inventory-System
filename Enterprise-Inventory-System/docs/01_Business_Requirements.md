# Project Overview

ABC Retail Pvt Ltd is a retail organization that purchases products from suppliers, stores them in warehouses, and sells them to customers.

Currently, the company manages inventory using Excel sheets, which leads to duplicate records, inaccurate stock levels, and no audit history.

The objective of this project is to build an enterprise-grade Inventory Management System using Microsoft SQL Server.

---

# Business Objectives

- Centralize inventory management.
- Maintain accurate stock across warehouses.
- Track purchases from suppliers.
- Track customer sales.
- Maintain complete audit history.
- Generate business reports.
- Improve data integrity.
- Support future integration with Python and AI.
