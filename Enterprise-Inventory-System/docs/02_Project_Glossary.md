# Project Glossary

| Business Term | Description |
|---------------|-------------|
| Product | Item sold by the company. |
| Category | Group to which a product belongs. |
| Supplier | Company supplying products. |
| Warehouse | Physical location where inventory is stored. |
| Inventory | Quantity of products currently available. |
| Customer | Person or company purchasing products. |
| Purchase Order | Order created to purchase products from suppliers. |
| Sales Order | Order created when a customer purchases products. |
| Purchase Order Item | Individual product within a purchase order. |
| Sales Order Item | Individual product within a sales order. |
| Stock Transfer | Movement of inventory between warehouses. |
| Audit Log | History of database changes. |
| Reorder Level | Minimum stock level before purchasing again. |
| Unit Price | Selling price of a product. |
| Cost Price | Purchase price of a product. |
| Available Stock | Stock available for sale. |
| Reserved Stock | Stock allocated to customer orders. |
| Active | Record currently in use. |
| Inactive | Record no longer available for business operations. |
